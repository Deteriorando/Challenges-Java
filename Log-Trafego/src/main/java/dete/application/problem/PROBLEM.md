# Analisador de log de Tráfego desafio

Tem que fazer uma lista de entradas de log onde cada entrada
contém:

- IP de origem (String)
- Tempo de resposta. (Long)
- Status da requisição (Enum)

Fazer uma classe base para os logs de entradas, com os atributos
acima.

Criar uma classe para processar os dados que mantenha um ArrayList.
  
  
O Processador: Crie uma classe LogProcessor que mantenha um ```ArrayList<LogEntry>.```

# Definir regras de négocio

- Filtragem dinâmica
- Cálculo de perfomace.
- Detecção de Anomalias
- Ordenação customizada.
